# DigiTools

## 📌 Description

DigiTools is a simple web application where users can see digital products and add them to a cart.

## ⚙️ Technologies

- React.js
- Tailwind CSS
- DaisyUI
- JavaScript
- React-Toastify

## ✨ Features

- Show products
- Add to cart
- Remove from cart
- Show total price
- Toast message (add/remove)

## 🚀 Run Project

npm install
npm run dev
